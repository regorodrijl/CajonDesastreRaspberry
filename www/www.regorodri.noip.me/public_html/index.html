Proyectos propios

"Proyecto Claustro"

Se trata de un proyecto de fin de ciclo (DAW). El cual trata la gestión de Claustros de profesores, la aplicación se conecta a un servidor LDAP para obtener los datos necesarios para dicha gestión. Se puede crear, modificar y borrar claustros. Para finalizar, la aplicación guarda la asistencia de los profesores a los claustros mediante la firma de estos, esto se lleva a cabo con una aplicación en Android nativa.(Puedes ver el código en github , esta en desarrollo la aplicación para IOS)

"Proyecto Tienda en FireBase"

Se trata de una pequeña aplicación en FireBase, la cual gestionan productos a modo de compra venta. Los usuarios se deben loguear. (Puedes ver el código en github)

"Proyecto Chat en FireBase"

Se trata de una pequeña aplicación en FireBase, la cual gestiona un chat entre la conserjería de un centro educativo y los profesores para registrar las posibles necesidades/incidencias que surjan. Para los profesores debes registrarte, para el Login de la conserjería: U-> conserje@iessanclemente.net , P-> abc123. (Puedes ver el código en github)

"Proyecto Bot en Telegram"

Se trata de un bot en telegram.(Puedes ver el código en github)

"Proyecto Inmobiliaria en Visual Basic"

Se trata de una aplicación en Visual Basic con la Base de Datos en MySql, la cual gestiona una inmobiliaria con sus altas, bajas y modificaciones de inmuebles. También gestiona las visitas a dichos inmuebles realizadas por los trabajadores y clientes de la inmobiliaria, así como el trámite de la documentación oportuna para un alquiler/compra. (Puedes ver el código en github)

"Proyecto Gestion Productos Empresa"

Se trata de una aplicación en Visual Basic, que gestiona los pedidos de una empresa entre esta y sus clientes. (Puedes ver el código en github)

"Proyecto para una empresa Extranjera (ELECTROSAN)"
